How It Works

Get up and running in minutes

1

Install & Configure

Run the installer to set up hooks, create directories, and configure Claude Code. The installation automatically handles all configuration including Claude Code settings.

2

Import Existing Conversations

Import all your existing Claude Code conversations into a searchable SQLite database. The database indexes messages, tool uses, and metadata for instant access.

3

Launch Dashboard

Start the Streamlit dashboard to browse sessions, search conversations, view analytics, and run AI-powered analysis across your entire conversation history.

4

Automatic Capture

From now on, every conversation is automatically captured when you exit Claude Code. Your insights database grows continuously without any manual intervention.
